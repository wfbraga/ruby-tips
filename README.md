# Ruby Tips

Purpose
- This repository helps keep my mind sharp by revisiting Ruby from time to time and capturing small pieces of knowledge about the language.

Contents
- [escape_characters.rb](escape_characters.rb) — examples of quoting and escape sequences.
- [the_p_method.rb](the_p_method.rb) — shows differences between `p` and `puts`.
- [the_print_method.rb](the_print_method.rb) — demonstrates `print` behavior.
- [the_puts_method.rb](the_puts_method.rb) — demonstrates `puts` usage and multiple args.
- [comments.rb](comments.rb) — tips for multiline comments in Ruby.

Usage
- Run any example with: `ruby filename.rb`

Notes
- Short, example-focused snippets meant for quick refreshers; contributions and personal notes welcome.
